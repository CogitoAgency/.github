## Products

Three macOS apps. All local. No accounts.

| Product | Job | Link |
|---------|-----|------|
| **BreathFlow** | System-wide voice-to-text. Hold a hotkey, speak, release — text lands in any app. On-device WhisperKit. No cloud. $49 one-time. | [cogito.cv/breathflow](https://cogito.cv/breathflow) |
| **Breathe** | Local memory for Claude Code. Auto-captures every session. Surfaces patterns. Restores context across projects. | [cogito.cv/breathe](https://cogito.cv/breathe) |
| **Keepr** | Relationship memory. Scans iMessage and Contacts locally, finds the people you went quiet with, opens the thread when it is time. | [cogito.cv/keepr](https://cogito.cv/keepr) |
